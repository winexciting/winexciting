# Winexciting · winexciting.com

Sito vetrina di Winexciting — Endless Passion.

## Come aggiornare i contenuti
Tutti i testi (tour, regioni, video) sono negli array all'inizio di `src/App.jsx`.
Le foto vanno in `public/images/`.

## Sviluppo locale (opzionale)
npm install
npm run dev

## Deploy
Il sito è pubblicato su Cloudflare Pages, collegato a questo repository.
Build command: `npm run build` — Output: `dist`
Ogni push su `main` pubblica automaticamente la nuova versione.
